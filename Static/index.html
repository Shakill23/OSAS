OSAS API Documentation

Functionality URL Link Method
Fetch all Carts (Admin only) http://localhost:2307/carts GET
Get Cart Items for User http://localhost:2307/carts/user/:id/carts GET
Add to Cart http://localhost:2307/carts/user/:id/cart POST
Edit Cart Item http://localhost:2307/carts/user/:id/cart/:cartItemId PATCH
Delete All Items from Cart http://localhost:2307/carts/user/:id/cart DELETE
Delete Specific Item from Cart http://localhost:2307/carts/user/:id/cart/:cartItemId DELETE
Get All Users http://localhost:2307/users GET
Get User by ID http://localhost:2307/users/:id GET
Register User http://localhost:2307/users/register POST
Login User http://localhost:2307/users/login POST
Edit User http://localhost:2307/users/:id PATCH
Delete User http://localhost:2307/users/:id DELETE
Get All Products http://localhost:2307/products GET
Get Product by ID http://localhost:2307/products/:id GET
Add Product http://localhost:2307/products POST
Edit Product http://localhost:2307/products/:id PATCH
Delete Product http://localhost:2307/products/:id DELETE
